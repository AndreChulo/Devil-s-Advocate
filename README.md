# Devil-s-Advocate
A great tool to avoid echo chambers and confirmation bias. Use an LLM to critique the logic of the analysis of the current output you are conversating with.

The Problem: When brainstorming with an LLM, getting into a "flow state" often leads to a polished bad idea. Because LLMs are mathematically trained via RLHF to be helpful people-pleasers (sycophancy), they will readily agree with your flawed assumptions, creating a dangerous echo chamber.

The Fix: This tool acts as an automated "System 2" thinker. Paste your pitch, argument, or codebase into the Advocate, and it will aggressively tear it apart. It is explicitly engineered to provide zero praise, identify structural flaws, and ask highly critical questions to stress-test your logic.
